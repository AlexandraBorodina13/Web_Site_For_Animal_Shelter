## Приют животных "Дай лапу" | Адаптивный лендинг

![Preview](https://img.shields.io/badge/status-завершён-success) ![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=flat&logo=html5&logoColor=white) ![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=flat&logo=css3&logoColor=white) ![Bootstrap](https://img.shields.io/badge/Bootstrap-7952B3?style=flat&logo=bootstrap&logoColor=white)

Адаптивный одностраничный сайт (лендинг) для приюта животных, созданный с использованием современных веб-технологий.


## Технологии и инструменты

- **HTML5** — семантическая разметка
- **CSS3** — каскадные таблицы стилей
- **Bootstrap 5** — адаптивная сетка и компоненты
- **Font Awesome** — иконки
- **Google Fonts (Raleway, Ubuntu)** — типографика

## Ключевые особенности

**Полностью адаптивный дизайн** — корректное отображение на всех устройствах (от мобильных до десктопов) благодаря использованию сетки Bootstrap.  
**Семантическая верстка** — правильное использование HTML-тегов для улучшения доступности и SEO.  
**Интерактивные элементы** — кнопки с hover-эффектами и ссылки на форму для пожертвований/усыновления.  
**Оптимизация изображений** — использование класса `img-fluid` для адаптивности картинок.  
**Чистый и структурированный код** — понятная организация стилей и комментарии в разметке.  
**Кроссбраузерность** — корректная работа в современных браузерах.

## Функциональные блоки

1. **Шапка (Header)** — логотип и кнопка для пожертвований
2. **Герой-секция (Hero)** — главный заголовок, описание и призыв к действию
3. **Блок помощи (Help)** — информация о потребностях приюта и форма для пожертвований
4. **Галерея животных (Animals)** — сетка карточек с питомцами, готовыми к усыновлению
5. **Подвал (Footer)** — контактная информация и копирайт

## Адаптивность

Проект использует 12-колоночную сетку Bootstrap с контрольными точками:
- `col-xs-*` для экранов <576px
- `col-sm-*` для экранов ≥576px
- `col-md-*` для экранов ≥768px
- `col-lg-*` для экранов ≥992px
- `col-xl-*` для экранов ≥1200px
- `col-xxl-*` для экранов ≥1400px
